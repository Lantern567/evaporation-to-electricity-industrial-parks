# Manuscript Outline

## Title

Evaporation-to-electricity transition: 蒸发变化对工业能源与零碳园区的影响研究

## 1. Introduction

- 自然蒸发是高盐废水、浓盐水减量和 ZLD 系统中的隐性能源服务。
- 蒸发能力下降会把自然消纳转移到电力、蒸汽、设备、土地和盐渣处置。
- 零碳园区同时需要更多绿电支撑低碳产业和新增水处理负荷，形成水-能-碳耦合约束。

## 2. Conceptual Framework

- 自然蒸发作为可替代生产要素。
- 从蒸发缺口到能源缺口：`DeltaV -> DeltaElec / DeltaSteam -> Cost / CO2`。
- 从企业和行业影响扩展到园区源荷匹配与省级资源配置。

## 3. Result 1

蒸发能力变化已经显著改变我国工业废水处理的经济成本与碳排放。

## 4. Result 2

“自然消纳”向“电力驱动”的转移会冲击零碳园区的源荷匹配与设计冗余。

## 5. Result 3

学习曲线和零碳园区扩张将改变最优资源配置，省级协同方案可同时降低成本和碳排。

## 6. Discussion

自然蒸发定价为工业系统中的气候敏感型能源服务，是零碳园区基础设施设计的重要补充。

## 7. Methods

- 蒸发亏缺指数构造。
- 高盐行业暴露度和浓盐水矩阵。
- 双向固定效应、事件研究和稳健性检验。
- 工程放大和全国核算。
- 零碳园区小时级压力测试。
- 省级两阶段随机优化。
