# Result Framework

## Result 1

蒸发能力变化已经显著改变我国工业废水处理的经济成本与碳排放。

### 1.1 先证明“效应显著”

蒸发亏缺会使高盐废水暴露行业的处理能耗、运行成本和碳排上升。

- 构建县域或园区尺度蒸发亏缺指数 `EDI`，将气象站逐日蒸发量、风速、湿度、辐射和降水异常匹配到工业企业、排污许可和园区位置。
- 用双向固定效应、事件研究和高盐行业暴露度交互项识别蒸发冲击的因果影响。
- 将煤化工、火电脱硫废水、石化化工、盐化工、印染、电镀、制药、光伏材料等作为高暴露组，普通低盐工业废水行业作为对照组。

### 1.2 再测量“影响大小”

全国增量成本和增量碳排主要集中在少数省份、行业和园区类型。

- 建立省份-行业-园区浓盐水产生量矩阵。
- 将缺失自然蒸发量折算为扩建蒸发池面积、机械蒸发电耗、蒸汽或余热需求、设备折旧和碳排。
- 输出全国、分省、分行业年度成本增加量、碳排增加量和风险排名。

## Result 2

“自然消纳”向“电力驱动”的转移会冲击零碳园区的源荷匹配与设计冗余。

### 2.1 蒸发量下降会形成连续性用电负荷

- 将蒸发亏缺水量转化为 MVR、低温蒸发、膜浓缩和结晶系统的等效电力负荷。
- 构造小时负荷曲线，评估对园区绿电自给率、峰值负荷、储能规模和碳强度约束的冲击。
- 在 5%、10%、15% 蒸发能力下降情景下，识别绿电匹配不足、备用容量不足或碳排核算不达标的园区。

### 2.2 给出园区规划的最低冗余规则

- 定义园区级水-能-碳安全约束：绿电供应需要以目标可靠性覆盖工业负荷和污水电驱负荷。
- 反推最小绿电冗余率、设备冗余率和应急处理能力。
- 按西北煤化工/盐化工园区、华北火电-化工园区、东部沿海高端制造园区、南方高湿低蒸发园区形成工程设计建议。

## Result 3

学习曲线和零碳园区扩张将改变最优资源配置，省级协同方案可同时降低成本和碳排。

### 3.1 技术学习和绿电成本决定碳排结果

- 引入 MVR、低温蒸发、膜浓缩、电渗析、热泵、储能、光伏、风电和余热利用的学习曲线。
- 构建 2025-2035 年技术成本、电耗和排放强度情景。
- 建立各省新增高盐废水处理需求、绿电可得性、土地约束、余热资源和电网碳因子的动态数据库。

### 3.2 通过省级优化配置确定最优组合

- 建立两阶段随机优化模型。
- 决策变量包括蒸发池面积、MVR 容量、膜浓缩容量、储能容量、绿电直连或外购绿电比例、余热利用规模和传统备用处理能力。
- 与传统方案比较经济成本、碳排、土地占用和供电可靠性的改进幅度。
