# Evaporation-to-Electricity Transition

蒸发变化对工业能源与零碳园区的影响研究。

本仓库支持“自然蒸发能力变化如何改变工业园区水处理能源成本、碳排放和零碳园区设计冗余”的研究。项目的核心判断是：自然蒸发不是单纯的气象变量，而是高盐废水、浓盐水减量和零排放系统中的一项隐性能源服务。当蒸发能力下降时，原本由土地、太阳辐射和风承担的自然消纳，会转移到 MVR、低温蒸发、膜浓缩、蒸发结晶、泵送、盐渣处置和绿电系统上。

## Research Question

本研究不只问“蒸发量下降会不会增加污水处理成本”，而是进一步追问：

- 当自然蒸发能力下降时，零碳园区是否会出现“减碳用能”和“治水用能”的竞争？
- 高盐废水处理新增电力、蒸汽、土地和设备需求会如何影响园区绿电自给率、储能规模和碳强度约束？
- 在省级尺度上，蒸发池、MVR、膜浓缩、余热、储能和外购绿电之间的最优配置如何变化？

## Result Framework

| Result | 核心问题 | 主要产出 |
|---|---|---|
| Result 1 | 蒸发能力变化是否已经显著改变工业废水处理成本、能耗和碳排 | 蒸发亏缺指数、行业暴露度、因果识别结果、全国分省分行业成本和碳排增量 |
| Result 2 | “自然消纳”向“电力驱动”的转移是否冲击零碳园区源荷匹配 | 园区小时级压力测试、设计失效概率、最低绿电和设备冗余规则 |
| Result 3 | 技术学习和零碳园区扩张下如何优化资源配置 | 2025-2035 情景数据库、省级两阶段随机优化、设备-电力-热-储协同方案 |

## Suggested Structure

- `data/`: 数据来源、数据字典、处理流程、脚本和可复现数据说明。大型原始数据不进入 Git。

- `literature/`: 文献检索策略、筛选记录、阅读笔记和证据表。
- `manuscript/`: 论文提纲、Result 框架、图表清单、段落草稿和修改记录。
