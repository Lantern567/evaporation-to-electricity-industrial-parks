# Technical Parameters

本文件用于记录工程核算和优化模型所需参数。每个参数都应保留来源、年份、单位、适用边界和不确定性范围。

| 参数 | 单位 | 低值 | 中值 | 高值 | 来源 | 备注 |
|---|---|---:|---:|---:|---|---|
| MVR 单位处理电耗 | kWh/t |  |  |  |  | 与进水 TDS、浓缩倍数和蒸发温度有关 |
| 低温蒸发单位处理电耗 | kWh/t |  |  |  |  | 需区分电驱和热驱边界 |
| 膜浓缩单位处理电耗 | kWh/t |  |  |  |  | 与预处理、回收率和污染控制有关 |
| 蒸发结晶单位处理电耗 | kWh/t |  |  |  |  | 可拆分电耗和热耗 |
| 蒸发池建设成本 | 元/m2 |  |  |  |  | 包括土地、防渗、土建和监测 |
| 盐渣处置成本 | 元/t |  |  |  |  | 区分一般固废和危废 |
| 储能 CAPEX | 元/kWh |  |  |  |  | 用于园区压力测试和优化 |
| 绿电价格 | 元/kWh |  |  |  |  | 区分直连、PPA、绿证和外购电 |
| 电网排放因子 | tCO2/MWh |  |  |  |  | 分省或区域电网 |
