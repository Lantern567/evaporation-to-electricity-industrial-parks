# Data Sources

本文档记录本项目拟使用的数据体系。当前版本根据本地研究方案整理，后续每次下载或替换数据时应更新 `data/docs/data_inventory.csv`。

## Source Summary

| 数据类别 | 典型来源 | 用途 |
|---|---|---|
| 气象蒸发数据 | 中国气象局国家气象信息中心逐日蒸发皿蒸发量、气温、风速、相对湿度、日照/辐射、降水；ERA5 或 CMFD 再分析数据 | 构建县域、园区和企业尺度蒸发亏缺指数 EDI |
| 工业废水与高盐暴露数据 | 生态环境统计年报、全国排污许可证、重点排污单位名录、行业产量、火电装机、煤化工和盐化工项目清单、园区环评与零排放项目案例 | 构建省份-行业-园区浓盐水产生量矩阵和高盐暴露度 |
| 能源和碳排数据 | 分省电价、分时电价、绿电交易价格、电网排放因子、煤/气/蒸汽价格、工业余热资源、园区光伏/风电/储能装机潜力 | 将蒸发缺口转化为电力、蒸汽、成本和碳排 |
| 零碳园区数据 | 国家级零碳园区名单、省级零碳/低碳园区试点、园区产业结构、用电负荷、绿电供应方式、能碳管理指标 | 开展园区源荷匹配压力测试和最低冗余规则计算 |
| 技术成本数据 | 自然蒸发池土地、防渗、土建和运维成本；MVR、低温蒸发、蒸发结晶、膜浓缩、电渗析、储能和余热利用的 CAPEX、OPEX、单位能耗和寿命 | 支撑工程放大、全国核算和省级资源配置优化 |

## Data Handling Principles

- 原始数据保持不改动，按来源存放在 `data/raw/` 或 `data/external/`。
- 所有清洗、匹配和指标构造过程应由 `data/scripts/` 中的脚本复现。
- 对企业、园区和排污许可明细数据，应优先发布聚合结果、字段说明和处理方法，避免公开敏感明细。
- 对再分析气象数据和大型栅格数据，只提交下载说明、变量清单和空间范围，不提交大文件。

## Planned Data Products

| 输出表 | 说明 |
|---|---|
| `county_evaporation_deficit.csv` | 县域逐年或逐月蒸发亏缺指数、气象控制变量和异常指标 |
| `park_evaporation_deficit.csv` | 园区尺度蒸发亏缺指数和气象暴露指标 |
| `industry_high_salinity_exposure.csv` | 行业高盐废水暴露度、浓盐水系数和自然蒸发依赖比例 |
| `province_industry_brine_matrix.csv` | 省份-行业浓盐水产生量、自然蒸发依赖度和情景缺口 |
| `park_stress_test_results.csv` | 零碳园区压力测试、失效概率和最低冗余率 |
| `province_optimization_results.csv` | 省级设备、电力、余热、储能和外购绿电最优配置 |
