# Data

本目录用于管理蒸发变化、工业高盐废水、零碳园区、能源价格、碳排因子和技术成本数据的说明、字段字典、处理脚本与可复现流程。

大型原始数据、再分析数据、排污许可明细、企业点位和生成结果默认不进入 Git。仓库只跟踪数据清单、下载说明、处理逻辑和可公开的元数据。

## Folder Plan

- `docs/`: 数据源说明、变量字典、处理流程和数据清单。
- `scripts/`: 数据下载、清洗、空间匹配、指标构造和情景核算脚本。
- `raw/`: 原始数据本地存放位置，不纳入 Git。
- `external/`: 外部整理数据或手工下载数据，不纳入 Git。
- `interim/`: 标准化中间表，不纳入 Git。
- `processed/`: 分析就绪数据，不纳入 Git。
- `outputs/`: 地图、图表、QA 报告和模型输出，不纳入 Git。
